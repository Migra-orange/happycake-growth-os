# HappyCake Growth OS

Build a real business growth system for HappyCake Sugar Land, not a generic hackathon bot.

Rules:
- Runtime: Claude Code CLI via `claude -p` from local wrappers.
- Owner UI: Telegram only.
- Never use Claude Agent SDK, LangGraph, CrewAI, n8n, or another core LLM provider.
- Customer-facing language: English only.
- Brand spelling: HappyCake.
- Never fabricate prices, inventory, hours, allergens, delivery, or policies; use MCP/source-of-truth or owner approval.
- Ready-made classic cakes first; do not position as a custom-cake bakery.
- Evidence matters: every scenario should be logged.
